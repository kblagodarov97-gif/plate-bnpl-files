Внешний вид виджетов можно посмотреть открыв в браузере файл *example.html*

Для работы виджетов необходимо:
- подключить на страницы сайта файл стилей *bnpl.css* в теге `<head>`
и файл скриптов *bnpl.js* в самом низу страницы перед закрывающим тегом `</body>`
- скопировать блок `<div class="bnpl-modal">` вместе со всем его содержимым на каждую страницу, где будут использоваться виджеты. Он должен быть непосредственным потомком тега ```<body>```
- также необходимо скопировать файлы шрифтов из папки *fonts* и поменять пути к этим файлам в директивах *@font-face* в начале файла *bnpl.css*

Для использования нужного виджета необходимо скопировать его разметку из файла *example.html* и разместить в желаемом месте на странице. В коде *example.html* разметка которую нужно скопировать обозначена комментариями с номерами фреймов.

В разметке у каждого используемого виджета необходимо указать корректные значения четырех data-атрибутов:
- *data-price* - полная стоимость товара
- *data-payments-number* - количество платежей (частей)
- *data-first-payment-percents* - доля первого платежа (части), в процентах
- *data-payments-interval* - интервал между платежами (частями), в днях.

Значения из этих атрибутов используются вычисления и для динамической подстановки данных о сроках, количестве платежей и т.д. в тело виджета и модального окна.

Например:
- Для использования виджета "Виджет №1" нужно скопировать тег, находящийся между комментариями `<!-- Начало - Виджет №1 -->` и `<!-- Конец - Виджет №1 -->`, со всем его содержимым и разместить его в желаемом месте на странице. Указать корректные значения дата-атрибутов перечисленных выше.
- Для использования виджета с расписанием платежей, например "Виджет №6", нужно скопировать тег, находящийся между комментариями `<!-- Начало - Виджет №6 -->` и `<!-- Конец - Виджет №6 -->`, со всем его содержимым. Указать корректные значения дата-атрибутов перечисленных выше.

Чтобы сменить цвета черточек на расписании платежей и иконок в модальном окне нужно изменить значение css-переменных *bnpl-main-color*, *bnpl-main-color-light* и *bnpl-secondary-color* в файле *bnpl.css*

**Важное замечание:** виджеты *№3, №4, №5* возможно использовать только в том случае если все платежи равны между собой, т.е размер первого платежа равен каждому из последующих. Например, если общее количество платежей 5 и размер первого платежа 20%. Или общее количество платежей 4 и размер первого платежа 25%.
